A/B Testing - Anlise de Reteno no Jogo Cookie Cats

Objetivo

Este notebook tem como objetivo aplicar passo a passo os conceitos de A/B Testing (Teste A/B) para avaliar o impacto de uma alterao no design do jogo Cookie Cats, um jogo do tipo "match-3" (estilo Candy Crush) com ampla base de usurios. A mudana testada foi o deslocamento do primeiro "gate" (barreira no progresso do jogo) do nvel 30 (grupo controle) para o nvel 40 (grupo teste).

Nosso objetivo responder:

Mover o primeiro gate do nvel 30 para o nvel 40 afeta negativamente a reteno dos jogadores no 1 e 7 dia aps a instalao do jogo?

  • Gates so "barreiras de progresso" que interrompem temporariamente o jogador no jogo.

Sobre os Dados

O dataset contm informaes de 90.189 jogadores que instalaram o jogo durante o perodo do experimento A/B. Cada jogador foi aleatoriamente.

Nota: Todos os usurios baixaram a mesma verso do app Cookie Cats. A diferena entre gate_30 e gate_40 controlada por configurao de servidor em tempo real, ou seja, Cada jogador foi aleatoriamente alocado em um dos grupos.

  • gate_30: grupo controle (barreira no progresso do jogo: gate no nvel 30)
  • gate_40: grupo teste (barreira no progresso do jogo: gate no nvel 40)

As variveis disponveis so:

  • userid: identificador nico do jogador
  • version: grupo de alocao (gate_30 ou gate_40)
  • sum_gamerounds: total de rodadas jogadas na primeira semana
  • retention_1: jogador voltou no dia seguinte? (1 sim, 0 no)
  • retention_7: jogador voltou aps 7 dias? (1 sim, 0 no)

Etapas da Anlise

  1. Entendimento do problema de negcio e dos dados
  2. Limpeza e checagem dos dados: valores ausentes, outliers, valores inesperados
  3. Estatsticas descritivas e visualizaes
  4. Testes de hiptese:
    • Verificao de suposies (normalidade, homogeneidade de varincias)
    • Aplicao de testes estatsticos:
      • Shapiro-Wilk (normalidade)
      • Levene (homogeneidade de varincia)
      • T-Test / Welch Test (teste de diferenas de mdias)
      • Mann-Whitney U Test (caso no-normal)
  5. Avaliao dos resultados e inferncia
  6. Recomendaes para tomada de deciso

Testes Estatsticos:

1. Teste de Shapiro-Wilk

  • Objetivo: Verificar normalidade dos dados
  • H: Os dados seguem distribuio normal
  • H: Os dados no seguem distribuio normal
  • Deciso: Se p-valor > 0,05 Normal

2. Teste de Levene

  • Objetivo: Verificar homogeneidade de varincias
  • H: As varincias so iguais entre os grupos
  • H: As varincias so diferentes entre os grupos
  • Deciso: Se p-valor > 0,05 Homogneo

3. Teste T (Student)

  • Condies: Dados normais + varincias homogneas
  • H: No h diferena entre as mdias dos grupos
  • H: H diferena entre as mdias dos grupos

4. Teste de Welch

  • Condies: Dados normais + varincias no homogneas
  • H: No h diferena entre as mdias dos grupos
  • H: H diferena entre as mdias dos grupos

5. Mann Whitney U

  • Condies: Dados no normais (no paramtrico)
  • H: No h diferena entre as distribuies dos grupos
  • H: H diferena entre as distribuies dos grupos

  • = 0,05 (5% de chance de erro tipo I)
  • Deciso: Se p-valor 0,05 Rejeitar H (diferena significativa)

# Funo de Teste A/B - Soluo Automatizada
def teste_ab(dataframe, grupo, variavel_target):
    """
    Executa teste A/B automatizado seguindo premissas estatsticas

    Parmetros:
    - dataframe: DataFrame com os dados
    - grupo: nome da coluna com grupos A/B
    - variavel_target: nome da coluna a ser testada
    """

    # 1. SEPARAR GRUPOS A E B
    grupo_a = dataframe[dataframe[grupo] == "A"][variavel_target]
    grupo_b = dataframe[dataframe[grupo] == "B"][variavel_target]

    # 2. TESTE DE NORMALIDADE (Shapiro-Wilk)
    # H0: Distribuio  normal | H1: Distribuio no  normal
    normal_a = shapiro(grupo_a)[1] >= 0.05  # True = normal
    normal_b = shapiro(grupo_b)[1] >= 0.05  # True = normal

    # 3. DECISO DO TESTE ESTATSTICO
    if normal_a and normal_b:  # Ambos grupos so normais
        # TESTE PARAMTRICO

        # Teste de homogeneidade (Levene)
        # H0: Varincias iguais | H1: Varincias diferentes
        homogeneo = levene(grupo_a, grupo_b)[1] >= 0.05

        if homogeneo:
            # Teste T com varincias iguais
            p_valor = ttest_ind(grupo_a, grupo_b, equal_var=True)[1]
            tipo_teste = "Teste T (varincias iguais)"
        else:
            # Teste de Welch (varincias diferentes)
            p_valor = ttest_ind(grupo_a, grupo_b, equal_var=False)[1]
            tipo_teste = "Teste de Welch (varincias diferentes)"

    else:
        # TESTE NO-PARAMTRICO
        # Mann-Whitney U
        p_valor = mannwhitneyu(grupo_a, grupo_b)[1]
        tipo_teste = "Mann-Whitney U (no-paramtrico)"
        homogeneo = None  # No aplicvel

    # 4. INTERPRETAO DOS RESULTADOS
    significativo = p_valor < 0.05
    decisao = "Rejeitar H0" if significativo else "No rejeitar H0"
    interpretacao = "Grupos so DIFERENTES" if significativo else "Grupos so SIMILARES"

    # 5. RESULTADOS ORGANIZADOS
    resultados = pd.DataFrame({
        "Tipo_Teste": [tipo_teste],
        "P_Valor": [round(p_valor, 6)],
        "Significativo": [significativo],
        "Decisao": [decisao],
        "Interpretacao": [interpretacao]
    })

    # Adicionar homogeneidade se teste paramtrico
    if normal_a and normal_b:
        resultados["Homogeneidade"] = ["Sim" if homogeneo else "No"]
        resultados = resultados[["Tipo_Teste", "Homogeneidade", "P_Valor",
                               "Significativo", "Decisao", "Interpretacao"]]

    # 6. IMPRIMIR HIPTESES
    print("TESTE A/B - HIPTESES:")
    print("H0: Grupo A = Grupo B (no h diferena)")
    print("H1: Grupo A  Grupo B (h diferena)")
    print("Nvel de significncia:  = 0,05")
    print("=" * 50)

    return resultados

def teste_retencao(dataframe, grupo, variavel_retencao, nome_teste):
    """Testa significncia estatstica para variveis de reteno (propores)"""

    # Contar sucessos (retidos) por grupo
    tabela_contingencia = pd.crosstab(dataframe[grupo], dataframe[variavel_retencao])

    # Preparar dados para teste Z de propores
    grupo_a = dataframe[dataframe[grupo] == "A"]
    grupo_b = dataframe[dataframe[grupo] == "B"]

    # Contar sucessos e totais
    sucessos_a = grupo_a[variavel_retencao].sum()
    total_a = len(grupo_a)
    sucessos_b = grupo_b[variavel_retencao].sum()
    total_b = len(grupo_b)

    # Implementao manual do teste Z para duas propores
    p1 = sucessos_a / total_a
    p2 = sucessos_b / total_b

    # Proporo combinada
    p_combined = (sucessos_a + sucessos_b) / (total_a + total_b)

    # Erro padro
    se = np.sqrt(p_combined * (1 - p_combined) * (1/total_a + 1/total_b))

    # Z statistic
    z_stat = (p2 - p1) / se if se > 0 else 0

    # P-valor (teste bicaudal)
    p_valor = 2 * (1 - stats.norm.cdf(abs(z_stat)))

    # Teste Chi-quadrado como validao
    chi2, p_chi2, dof, expected = chi2_contingency(tabela_contingencia)

    # Calcular propores
    prop_a = sucessos_a / total_a
    prop_b = sucessos_b / total_b
    diferenca = prop_b - prop_a

    # Resultados
    print(f"\n{nome_teste}:")
    print(f"  Grupo A: {prop_a:.4f} ({prop_a*100:.2f}%)")
    print(f"  Grupo B: {prop_b:.4f} ({prop_b*100:.2f}%)")
    print(f"  Diferena: {diferenca:+.4f} ({diferenca*100:+.2f}pp)")
    print(f"  Z-statistic: {z_stat:.4f}")
    print(f"  P-valor (Z-test): {p_valor:.6f}")
    print(f"  P-valor (Chi): {p_chi2:.6f}")
    print(f"  Significativo (=0.05): {'SIM' if p_valor < 0.05 else 'NO'}")

    return {
        'teste': nome_teste,
        'prop_a': prop_a,
        'prop_b': prop_b,
        'diferenca': diferenca,
        'z_stat': z_stat,
        'p_valor': p_valor,
        'significativo': p_valor < 0.05
    }
